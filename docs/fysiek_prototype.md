# Fysiek prototype

De gebruikte microcontroller voor dit project is een [NodeMCU 32S](/imgs/nodemcu.jpg). Hierbij zijn de pinnen GPIO16 en GPIO17 de TX en RX pinnen. Deze moeten elk verbonden worden met de Rx en Tx pin van de [DFPlayer mini](/imgs/dfpin.jpg).
