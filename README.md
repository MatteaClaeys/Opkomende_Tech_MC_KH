# Opdracht opkomende technologieën

Dit project wordt gecombineerd met het project *Gebruiksgericht Ontwerpen*. In dat project wordt een product ontwikkeld dat mensen met een visuele beperking ondersteunt bij het verwerken van muziekpartituren. Dit gebeurt aan de hand van audiopartituren die met de voeten kunnen worden bediend (pauzeren, terugspoelen, ...). Binnen dit project worden de elektronica en de code voor dat systeem ontworpen. ([Project audio partituren](https://github.com/KobeHolderbeke/Visually-Impaired))

De lijst met benodigdheden kan teruggevonden worden in [Requirements](docs/Requirements.md).

De opbouw van het project staat bescshreven in [Prototype](docs/fysiek_prototype.md).
